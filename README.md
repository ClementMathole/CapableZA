# CapableZA: Skills Audit System ![ASP.NET Core MVC](https://img.shields.io/badge/ASP.NET%20Core%20MVC-512BD4?style=flat&logo=.net&logoColor=white)

### A comprehensive Skills Management & Auditing platform built with ASP.NET Core and Firebase.

> **Manage. Track. Verify.**

Capableza is a complete **Skills Audit and Employee Management System**, designed to streamline the process of tracking, validating, and reporting employee competencies.

Built with a **clean, role-based ASP.NET Core MVC architecture** and powered by **Firebase** for real-time data and secure authentication.
